# 个人作品展示网站 — 设计文档

## 概述

基于 reveal.js 的全屏幻灯片式个人作品集网站，面向 AI/算法工程师岗位面试场景。风格创意活泼，内容全面展示个人信息、AI 项目经历、作品演示和技术亮点。

## 技术选型

- **框架**：reveal.js 5.x（CDN 引入）
- **部署**：单个 HTML 文件（`portfolio.html`），可部署到 Netlify
- **依赖**：无需构建工具，reveal.js + highlight.js 通过 CDN 加载

## 幻灯片结构

reveal.js 支持水平（章节间）和垂直（章节内子页）导航。

### 第 1 章：封面（1 页）

- 名字：Garra
- 一句话定位："AI Engineer / Creative Developer"
- 动态背景：CSS 渐变动画 + 浮动粒子效果
- 底部提示："按 → 或空格继续"

### 第 2 章：关于我（2 页）

**页 2.1 — 个人简介**
- 简短自我介绍（3-4 句话）
- 关键标签：Python / PyTorch / NLP / CV / Full-Stack / Game Dev

**页 2.2 — 技术栈**
- 用 CSS 实现的技能雷达图或进度条

- 每项配图标 + 熟练度指示

### 第 3 章：AI 项目经历（4 页）

## 视觉设计

### 配色方案

创意活泼风格：
- **主背景**：深蓝紫渐变（#1a0533 → #0d1b2a）
- **主色调**：亮紫 #a855f7 + 青色 #06b6d4
- **强调色**：粉色 #ec4899、橙色 #f97316（用于标签和高亮）
- **文字**：白色 #ffffff、浅灰 #94a3b8

### 动画与过渡

- 幻灯片切换：slide（水平）+ fade（垂直）
- Fragment 动画：fade-up 逐步显现
- 封面：CSS 渐变动画循环 + 浮动光点
- 代码块：打字机效果（可选）

### 排版

- 标题：系统字体，大号加粗
- 正文：16-20px，行高 1.6
- 代码：Fira Code / monospace

## 响应式适配

- 桌面端：全屏演示，最佳体验
- 平板/手机：支持触摸滑动翻页，字体自动缩放
- reveal.js 内置响应式支持

## 文件结构

```
portfolio.html    — 单文件，包含所有内容和样式
```

所有 reveal.js 和 highlight.js 依赖通过 CDN 加载，无需额外文件。
