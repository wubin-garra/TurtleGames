# 游戏库首页设计规格

## 概述

为 Garra 的乌龟游戏系列设计一个 PS5 风格的游戏库首页，作为统一入口展示和启动所有游戏。

## 游戏列表

| 游戏 | 文件 | 图标 | 描述 | 标签 |
|------|------|------|------|------|
| 乌龟赛跑 | `turtle-race.html` | 🏁 | 装备你的乌龟，征战排位赛，成为赛道之王 | 竞速、排位赛、装备系统 |
| 合成大乌龟 | `merge-turtle.html` | 🐢 | 合成各种乌龟，解锁稀有品种，探索海洋世界 | 合成、海洋主题、收集 |

## 布局结构

采用 PS5 经典横向焦点布局，单个 HTML 文件 `index.html`，全屏三段式：

```
┌──────────────────────────────────────────┐
│  顶栏: Logo "GARRA"  |  导航  |  时钟 头像  │
├──────────────────────────────────────────┤
│                                          │
│              英雄区 (flex:1)               │
│         辉光背景 + 氛围粒子                  │
│           游戏图标 (大)                     │
│           游戏名称 (大字)                   │
│           一句话描述                        │
│           标签 (pills)                    │
│           [▶ 开始游戏] 按钮                 │
│                                          │
├──────────────────────────────────────────┤
│  底部选择栏: [🏁 乌龟赛跑] [🐢 合成大乌龟]   │
└──────────────────────────────────────────┘
```

## 视觉风格

- **配色**：PS5 深蓝经典
  - 背景渐变：`#0a1628` → `#060e1a` → `#000`
  - 主色调：`#0070d8`（PS蓝）
  - 辅助色：`#00b4d8`（亮蓝）
  - 文字：`#fff`（标题）、`#6688aa`（描述）、`#556`（次要）
- **氛围**：微型粒子点缀，慢速浮动动画
- **辉光**：英雄区中心径向渐变，随游戏切换变色（蓝/绿）
- **毛玻璃**：底部选择栏 `backdrop-filter: blur(20px)`

## 交互设计

### 游戏切换
- 点击底部缩略图切换当前游戏
- 英雄区内容淡入淡出过渡（opacity + translateY，250ms）
- 辉光颜色随游戏切换
- 选中缩略图放大 1.15x，白色边框，发光阴影

### 键盘支持
- 左右方向键切换游戏

### 开始游戏
- 点击"开始游戏"按钮，`window.location.href` 跳转到对应 HTML 文件
- 按钮 hover 上移 + 增强阴影

## 组件细节

### 顶栏 (56px)
- 左：Logo `🎮 GARRA`，导航链接（游戏库 active、最近游玩 disabled）
- 右：实时时钟 HH:MM，头像圆形 "G"

### 英雄区
- 游戏图标：80px emoji，带 drop-shadow
- 标题：42px，font-weight 900
- 描述：15px，`#6688aa`
- 标签：圆角 pills，半透明蓝底
- 按钮：渐变蓝，圆角 12px，48px 左右内边距

### 底部选择栏 (110px)
- 居中排列缩略图
- 缩略图：76x76px 圆角方块，渐变背景，emoji + 小字名称
- 顶部有半透明蓝色分隔线

## 技术要求

- 单个 `index.html` 文件，无外部依赖
- 纯 HTML/CSS/JS，内联所有样式和脚本
- 字体：系统字体栈（`-apple-system, 'Segoe UI', sans-serif`）
- 全屏布局，100vw x 100vh，overflow hidden
- 移动端响应式适配（缩小间距和字号）
- 游戏数据以 JS 数组存储，便于后续扩展

## 文件输出

- `index.html` — 游戏库首页
