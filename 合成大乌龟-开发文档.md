# 🐢 合成大乌龟 — 开发文档

> 女儿养的小乌龟去世了，她很伤心。于是爸爸做了这个游戏，让小乌龟们在游戏里永远活着，还能不断进化变强 💚

**线上体验:** https://merge-turtle.netlify.app/
**开发时间:** 2026年3月24日，约 1 小时
**技术栈:** 纯原生 HTML5 Canvas + JavaScript，零依赖，单文件 3245 行
**作者:** Garra × Claude

## 一、游戏概述

类似「合成大西瓜」的合成类休闲游戏，玩家从顶部掉落乌龟，相同种类碰撞自动合成进化为更高级的乌龟。乌龟堆叠超过警戒线 2 秒则游戏结束。目标是尽可能合成出最高级的乌龟，获得最高分。

## 三、核心系统

### 3.1 物理引擎（自研）

没有使用 Box2D / Matter.js 等第三方库，完全手写的圆形刚体物理引擎：

- **固定时间步模拟** — 1/60s（困难模式 1/120s 双精度）
- **碰撞检测** — O(n²) 直接检测 + 空间哈希网格加速（>20 物体时自动切换）
- **冲量求解** — 法线冲量 + 切线摩擦力，支持大小差异自适应摩擦增强
- **位置修正** — Baumgarte stabilization，带 slop 容差防止抖动
- **休眠系统** — 低速持续一段时间后休眠，碰撞自动唤醒，节省计算
- **防穿透** — 速度钳制（最大 600-800 px/s）
- **低速阻尼** — 接近静止时额外阻尼，加速稳定收敛

### 3.2 心情/表情系统（搞怪模式专有）

每只乌龟实时检测自身状态，显示对应的动态表情覆盖：

| 心情 | 触发条件 | 表现 |
|------|----------|------|
| 😵 falling | 刚掉落、速度快 | 螺旋晕眼 + 上方运动线 |
| 💫 impact | 落地瞬间 | 旋转星星环 + 烟雾 puff |
| 💕 love | 靠近同级乌龟 | 飘浮爱心 + 加重腮红 |
| 😰 squeezed | 被 3+ 个乌龟挤压 | 汗滴滴落 + 放射应力线 |
| ❗ panic | 接近警戒线 | 感叹号抖动 + 脸部泛红 |
| 🎉 happy | 合成成功后 2 秒 | 金色光环脉冲 + 彩虹环爆发 + 彩色 sparkle 旋转 + 音符飘浮 |
| 😴 sleepy | 静止超过 10 秒 | Zzz 飘浮 + 鼻涕泡（被撞会醒来） |

### 3.3 音效系统

全部使用 Web Audio API 程序化合成，无音频文件加载：

| 事件 | 音效设计 |
|------|----------|
| 掉落 | 400→200Hz 指数下降扫频，0.1s |
| 普通合成 | 基频 300+level×80 Hz 上升，chain>1 加三角波和声 |
| 铜牌合成(Lv6) | G-C 双音阶上升 |
| 银牌合成(Lv7) | G-C-E 三音阶上升 |
| 金牌合成(Lv8) | C-E-G-C 大和弦 + 0.4s 后三角波和声层 |
| 游戏结束 | 300→80Hz 锯齿波下降，0.5s |

### 3.4 视觉主题 — 海洋水族箱

游戏区域设计为深海水族箱风格：

- **背景渐变** — 从深暗蓝 `#0a1628` 到海底蓝 `#133a5e` 四段渐变
- **沙质海底** — 带纹理点的沙色渐变层
- **装饰元素** — 4 块岩石 + 6 株珊瑚（圆/枝/扇三种形态）+ 7 根海草（带叶片摇曳动画）
- **游动小鱼** — 4 条不同颜色的小鱼来回游动，带波浪起伏
- **气泡上升** — 15 个气泡持续上浮，左右轻摆
- **水面光线** — 3 束从顶部射入的淡蓝光柱
- **玻璃墙壁** — 半透明渐变 + 细线边缘模拟鱼缸玻璃
- **渐进式警戒线** — 4 级预警视觉：
  - 远距离：极淡虚线
  - 接近：虚线变实，显示"注意"
  - 很近：闪烁加粗，显示"⚠ 危险！⚠"
  - 越线：红色实线 + "⚠ DANGER ⚠" + 红色渐变带

## 四、游戏功能

### 4.1 难度系统

| 难度 | 掉落范围 | 物理特征 |
|------|----------|----------|
| 简单 | Lv 0-6（7种） | 宽松：高弹性 0.3，低摩擦，高速度上限 800 |
| 普通 | Lv 0-5（6种） | 适中：弹性 0.25，中摩擦，速度上限 700 |
| 困难 | Lv 0-4（5种） | 严格：低弹性 0.15，高摩擦 0.8，速度上限 600，双精度物理 |

### 4.2 Combo 连击

- 1.5 秒内连续合成触发 combo
- 倍率：x2, x3, ... 最高 x8
- 连击时显示带颜色的 "COMBO x3!" 浮动文字
- 每级 combo 有不同颜色（黄→橙→红→紫→蓝）

### 4.3 成就系统（13 个）

| 图标 | 名称 | 条件 |
|:----:|------|------|
| 🐣 | 初次合成 | 完成第一次合成 |
| 🔥 | 三连击 | 达成 3 连击 |
| 💥 | 五连击 | 达成 5 连击 |
| 🌟 | 超级连击 | 达成 8 连击 |
| 💰 | 小有积蓄 | 单局 500 分 |
| 💎 | 龟界富豪 | 单局 2000 分 |
| 👑 | 分数之王 | 单局 5000 分 |
| 🌸 | 花开富贵 | 合成花龟 |
| 🦈 | 深海来客 | 合成海龟 |
| 🏔 | 陆地巨兽 | 合成象龟 |
| 🏆 | 终极合成 | 合成加拉帕戈斯象龟 |
| 🔄 | 合成达人 | 单局合成 20 次 |
| ⚡ | 合成大师 | 单局合成 50 次 |

成就解锁时弹出金色边框通知条，2.5 秒后淡出。游戏结束面板展示本局新解锁的成就。

### 4.4 排行榜

- 本地存储 Top 20 记录
- 每条记录：分数 / 最高乌龟(图标) / 最高龟数量 / 难度(彩色) / 日期
- 前三名显示 🥇🥈🥉，当前成绩高亮
- 支持清除记录

### 4.5 双渲染风格

底栏可切换：
- **经典** — 简洁的圆形 + 壳纹图案，无表情
- **搞怪** — 完整卡通造型：四肢、尾巴、壳高光、独特面部 + 实时心情表情

### 4.6 分享

游戏结束和菜单都有分享按钮，复制温馨文案到剪贴板：

## 六、技术实现要点

### 单文件架构
所有代码在一个 `merge-turtle.html` 中（3245 行，162KB），包含 HTML/CSS/JS，零外部依赖，方便部署和分享。吉祥物图片以 base64 内嵌。

### 响应式布局
根据设备宽高动态计算 `layout` 对象，最大 420×900px，自适应各种屏幕。

### 移动端适配
- touch 事件 + mouse 事件双支持
- `passive: false` 防止滚动
- `user-select: none` 防止文字选中
- `safe-area-inset` 适配刘海屏
- `roundRect` polyfill 兼容微信浏览器
- `navigator.vibrate` 触觉反馈（合成/连击/游戏结束分级震动）

### 数据持久化
全部使用 `localStorage`：
- `turtle_merge_high_score` — 最高分
- `turtle_merge_leaderboard` — 排行榜 JSON
- `turtle_merge_difficulty` — 难度偏好
- `turtle_merge_style` — 渲染风格
- `turtle_merge_achievements` — 已解锁成就
