  └──────┴──────┴────────┴──────────┴──────────────────┘

  晋级/降级规则

  - 每个段位有独立的胜/负计数器
  - 赢3场 → 晋级动画 → 进入下一段位，计数器重置
  - 输3场 → 降级提示 → 回到上一段位，计数器重置
  - 青铜不会降级（保底），王者不会晋级（最高）
  - 晋级时有华丽动画，降级时有鼓励提示

  AI对手装备

  - 青铜：AI无装备，最弱对手
  - 白银：AI随机穿1件普通装备（head/face/back/mount之一）
  - 黄金：2件普通装备
  - 铂金：3件装备（含1件稀有）
  - 钻石：4件装备 + 鞋子
  - 星耀：4件高级装备 + 匹配地形鞋
  - 王者：4件传说级装备 + 最佳鞋子

  游戏流程

  1. 主菜单点击"段位赛"
  2. 显示当前段位、战绩（x胜/x负）、入场费
  3. 扣除入场费 → 进入选龟/选赛道/比赛（复用现有流程）
  4. 比赛结束：赢了加1胜+奖励金币，输了加1负
  5. 触发晋级/降级时播放动画

  数据持久化

  在localStorage中新增保存：rank（当前段位）、rankWins（当前胜场）、rankLosses（当前负场）
