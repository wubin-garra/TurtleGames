界面结构（自上而下）

① 顶栏 — 当前分数 + 游戏标题 + 最高分
② 难度选择 — 简单/普通/困难，游戏开始前可切换
③ 预览区 — 显示下一个将出现的乌龟
④ 投放区 — 当前乌龟 + 虚线引导线，左右滑动控制位置
⑤ 游戏区 — 深色背景，乌龟在此堆叠和合成
⑥ Game Over 线 — 乌龟超过此线则游戏结束
⑦ 底栏 — 展示所有乌龟种类图鉴

交互方式

  • 滑动/拖动：左右移动当前乌龟位置
  • 点击/松手：释放乌龟，开始下落
  • 合成动画：两只相同乌龟碰撞时播放粒子特效 + 震动
  • 分数弹出：合成时在合成位置弹出 "+XX" 分数提示

难度配置（已修正 ✅）

难度 掉落范围 说明
简单 Lv.1-5 草龟~鳄龟，高级龟直接出现
普通 Lv.1-4 草龟~锦龟，中等挑战
困难 Lv.1-3 草龟~花龟，全靠合成上分
💡 难度逻辑：掉落等级范围越大 = 越容易出现高级龟 = 更简单。困难模式只掉小龟，必须一步步合成才能拿到高分。
